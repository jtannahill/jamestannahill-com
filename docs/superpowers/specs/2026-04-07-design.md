# jamestannahill.com Design Spec

## Decisions Summary

| Decision | Choice |
|---|---|
| Layout | Split: full-viewport dark hero → white body |
| Hero background | NYC lightning nightscape photo (Image 3) |
| Headshot | Formal blue suit photo (Image 2) — bio section |
| Load animation | Cinematic blur fade (blur→sharp, staggered 150ms per element) |
| Scroll animation | IntersectionObserver fade-up on all body sections/cards |
| Accent color | Amber `#c9882a` |
| Favicon | JT monogram, NHG Display Bold, SVG + PNG fallbacks |
| Contact form | SES Lambda via API Gateway (no Formspree) |
| RDLB video | Full-width brand reel section between bio and ventures, muted autoplay loop |
| IMG_0154 video | Contact/FAQs page background, muted autoplay loop |

## Page Structure

### Header
- Fixed, `bg-black/80 backdrop-blur-md`
- Logo: "James Tannahill" — NHG Display, 11px, tracking widest, white
- Nav links: ABOUT | THE AGENCY | RDLB | INSIGHTS | CONTACT (amber)
- Mobile: hamburger → full-screen dark overlay menu (`bg-black/95`), links centered, 48px tap targets

### Home page sections (in order)

#### 1. Hero
- `min-h-screen`, flex column, justify-end, `pb-20 px-14`
- Background: `public/hero-bg.jpg` (nightscape photo), `object-cover`, `brightness(0.75)`
- Overlay: `linear-gradient(to bottom, rgba(0,0,0,0.15), rgba(0,0,0,0.82))`
- Elements (all blur-fade animated, staggered):
  - Eyebrow: "NYC — Private Equity" — 10px, tracking 0.3em, `rgba(255,255,255,0.3)`
  - H1: "Operator. Investor. Builder." — NHG Display, full size
  - Tagline paragraph — 15px, `rgba(255,255,255,0.42)`
  - CTAs: amber "Get in Touch" + outline amber "Plocamium Holdings"
- Scroll indicator: right-aligned, amber gradient line + "Scroll" text

#### 2. Bio Section (white)
- `max-w-6xl mx-auto px-14 py-24`
- Two-column grid: `1fr 380px`
- Left: section label (amber), 3 bio paragraphs, education + memberships
- Right: headshot image (blue suit photo), `aspect-ratio: 3/4`, amber 3px bottom bar
- Scroll reveal: entire section fades up

#### 3. Competencies (white)
- `border-t border-border max-w-6xl mx-auto px-14 py-16`
- Amber section label
- 3×2 grid, `gap-px bg-border`
- Each card: `bg-white p-7 border-t-2 border-amber`
- Card label: NHG Display 11px
- Card body: 11px gray
- Scroll reveal: staggered 80ms per card

#### 4. Ventures (white)
- Same container as competencies
- 4×2 grid
- Cards: white, transparent top border → amber on hover
- "Visit →" in amber
- Scroll reveal: staggered 60ms per card

#### 5. RDLB Brand Reel (dark) — bottom of page
- Full-width, `bg-black`
- `<video>` tag: `autoplay muted loop playsinline`, `width: 100%`, `max-height: 80vh`, `object-fit: cover`
- Source: `RDLB-brand-reel.mp4` served from CloudFront (NOT in repo — 210MB)
- No controls, no overlay text — pure video
- Subtle amber `border-top: 1px solid rgba(201,136,42,0.2)` above and below
- Sits directly above the footer

### FAQs / Contact page

#### Background video
- `IMG_0154.mov` transcoded to `contact-bg.mp4`, hosted on S3/CloudFront
- Full-viewport video: `fixed`, `object-fit: cover`, `opacity: 0.12`, `z-index: 0`
- All page content sits above with `position: relative; z-index: 1`
- Dark overlay: `bg-black/60` on the page wrapper

#### Contact form
- Posts via `fetch()` to API Gateway → SES Lambda
- Fields: firstName, lastName, email, phone (optional), subject, message
- Focus states: `border-amber-600`
- Submit button: amber background
- Inline success/error state — no page reload

#### FAQ accordion
- `<details>/<summary>`, `+` rotates to amber when open

## Videos — Hosting & Delivery

Both videos served from S3 + CloudFront (same distribution as site):

| File | Source | S3 key | Notes |
|---|---|---|---|
| RDLB brand reel | `RDLB Brand Equity Capsule.mp4` | `videos/RDLB-brand-reel.mp4` | Upload directly to S3, 210MB |
| Contact background | `IMG_0154.mov` | `videos/contact-bg.mp4` | Transcode `.mov → .mp4` (H.264) first |

Transcode command:
```bash
ffmpeg -i "/Users/jamest/Downloads/IMG_0154.mov" \
  -c:v libx264 -crf 23 -preset slow \
  -c:a aac -b:a 128k \
  -movflags +faststart \
  /tmp/contact-bg.mp4
```

Upload:
```bash
aws s3 cp "/Users/jamest/Downloads/RDLB Brand Equity Capsule.mp4" \
  s3://jamestannahill.com-site/videos/RDLB-brand-reel.mp4 \
  --content-type video/mp4 --cache-control "public, max-age=31536000"

aws s3 cp /tmp/contact-bg.mp4 \
  s3://jamestannahill.com-site/videos/contact-bg.mp4 \
  --content-type video/mp4 --cache-control "public, max-age=31536000"
```

Video URLs in code use CloudFront domain: `https://d9ttkh3tz30f6.cloudfront.net/videos/`

## SES Lambda — Contact Form

New CDK stack: `JamesTannahillContactStack` in `cdk/lib/contact-stack.ts`

**Lambda** (`cdk/lambda/contact/handler.py`):
```python
# Receives: {firstName, lastName, email, phone, subject, message}
# Sends via SES to: web@jamestannahill.com
# Returns: {success: true} or {error: "..."}
```

**API Gateway:** HTTP API, POST `/contact`, CORS allow `https://www.jamestannahill.com`

**SES:** `web@jamestannahill.com` must be verified in SES (us-east-1). If still in sandbox, request production access.
